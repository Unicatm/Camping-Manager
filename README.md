# Camping Manager

## About the Project
**Camping Manager** is a web-based administrative dashboard designed to digitize and streamline the daily management processes of a camping business.

The application is intended **exclusively for camping administrators and staff**, focusing on simplifying internal operations rather than offering direct booking capabilities to customers.

---

## Core Features
- **Reservation Management** – register and manage bookings, including check-in/check-out dates, number of guests, and assigned camping spots  
- **Customer Management** – store and manage customer information and reservation history  
- **Reports & Statistics** – generate operational reports to support daily administration  

---

## Tech Stack
This project is built using the **MERN Stack**, chosen for its scalability and the ability to use JavaScript across the entire application:

- **MongoDB** – database  
- **Express.js** – backend framework  
- **React.js** – frontend library  
- **Node.js** – server runtime
